= Clusterinformationen und Vertragsdetails anzeigen
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Sie können Vertragsdetails für jeden Ihrer lokalen ONTAP -Cluster in der NetApp Konsole anzeigen. Wenn Sie noch keinen Cluster entdeckt haben, können Sie dies auch auf der Seite *Abonnements und Lizenzen* tun.

.Erforderliche Konsolenrolle:
Speicheradministrator oder Speicherbetrachter.link:https://docs.netapp.com/us-en/console-setup-admin/reference-iam-predefined-roles.html["Erfahren Sie mehr über die Zugriffsrollen der NetApp Konsole."^]
